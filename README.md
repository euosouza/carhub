# App CarHub

## Como rodar a aplicação

#### Backend:
Entrar na pasta do backend(server): ``cd server``

```bash
  npm i
  npm run dev
```

#### Frontend:
Entrar na pasta do frontend(client): ``cd client``

```bash
  npm i
  npm start
```

## Como rodar os testes:
Entrar na pasta do backend(server): ``cd server``

```bash
  npm run test
```
