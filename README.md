# App CarHub

## Como rodar a aplicação

#### Backend:
Entrar na pasta do backend server: ``cd server``

```bash
  npm i
  npx prisma generate
  npm run dev
```

#### Frontend:
Entrar na pasta do frontend server: ``cd client``

```bash
  npm i
  npm start
```
